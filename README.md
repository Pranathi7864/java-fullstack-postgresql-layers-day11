# Java Full Stack - PostgreSQL with Layered Architecture (Day 11)

This repository contains the Day 11 Java Full Stack training project using Spring Boot layered architecture with PostgreSQL database.

## Topics Covered

- Model Layer
- Controller Layer
- Service Layer
- Repository Layer
- PostgreSQL Database
- Spring Data JPA
- REST API Development
- Database Connectivity
- CRUD Operations
- Data Persistence

## Technologies Used

- Java
- Spring Boot
- Maven
- PostgreSQL
- Spring Data JPA
- IntelliJ IDEA
- Postman

## Description

This project demonstrates Spring Boot layered architecture using Model, Controller, Service, and Repository layers. PostgreSQL was connected with the Spring Boot application to store and manage data permanently.

## Outcome

Layered architecture with PostgreSQL database connectivity was implemented successfully using Spring Boot and Spring Data JPA.
